# Suhith Reddy — Portfolio

A high-performance, cinematic personal portfolio site built using Next.js, Tailwind CSS, Framer Motion, GSAP, and Lenis smooth scrolling.

## 🚀 Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
